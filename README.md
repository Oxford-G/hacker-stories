# Hacker-Stories

Hacker-Stories is a React project where users can manage and organize their news.


## Built With

- Javascript
- React
- Node
- JSX
- HTML
- CSS

## Getting Started

To get a local copy up and running follow these simple example steps.

- Clone the repo https://github.com/Oxford-G/hacker-stories.git
- cd into the root directory
- `npm install` - This will install all the packages in the package.json file
- `npm start` - This will run the app locally in the local environment
